# ‚ö° Bolt.nvim ‚ö°
## A Personalized Neovim Configuration

![Neovim Screenshot - Optional: Replace with a cool screenshot of your config in action!]

A fast, highly optimized, readable, and feature-scalable Neovim configuration built with Lua, focusing on a minimalistic and productive coding experience. Designed by Jesus Plasencia who values speed, efficiency, and a finely-tuned environment.

---

## üöÄ Features

*   **Facade:** A friendly interface using [alpha](https://github.com/folke/tokyonight.nvim).
*   **Colorscheme:** A clean and modern look colorscheme with the help of [tokyonight](https://github.com/folke/tokyonight.nvim).
*   **Packer Manager:** Leveraging [lazy.nvim](https://github.com/folke/lazy.nvim) for on-demand plugin installation.
*   **Treesitter:** Powered by a left-side-panel [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) to view a structured tree representation of your source code.
*   **Fuzzy Finding:** Quickly find files, buffers, and commands with [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) and minimal keymaps.
*   **Buffer Manager:** Navigate between buffers with [bufferline.nvim](https://github.com/akinsho/bufferline.nvim).
*   **Syntax Highlighting:** Highlight when yanking (copying) text with [kickstart-highlight-yank](https://github.com/nvim-lua/kickstart.nvim).
*   **Enhanced Git Integration:** Seamless Git operations directly within Neovim with [gitsigns](https://github.com/lewis6991/gitsigns.nvim) and [lazygit](https://github.com/kdheepak/lazygit.nvim).
*   **Custom Keybindings:** Ergonomic and efficient keymaps for common tasks (check `lua/config/keymaps.lua` to check all of them!).
*   **Intelligent Auto-Completion:** Smart code suggestions for a wide range of languages.
*   **Powerful LSP Integration:** Language Server Protocol support for enhanced development.

## üõ†Ô∏è Requirements

Before installing `Bolt.nvim`, ensure you have the following installed:

*   **`git`:** Must have for cloning the repository and even work with in your own projects. [Download Git](https://git-scm.com/downloads)
*   **Neovim v0.10.0+ (or latest stable):** Required for optimal performance and Lua features. [Setup Nvim]()
    *   Check your Neovim version with `:version`
*   **A Nerd Font (Optional but Recommended):** For proper display of icons and glyphs (e.g., [Fira Code Nerd Font](https://www.nerdfonts.com/font-downloads)).

## üì¶ Installation

**1. Backup your existing Neovim configuration (optional but recommended):**

```bash
mv ~/.config/nvim ~/.config/nvim_old_backup
mv ~/.local/share/nvim ~/.local/share/nvim_old_backup
```

**2. Clone this repository:**

```bash
git clone https://github.com/jesusplasencia/bolt.nvim ~/.config/nvim
```

**3. Launch Neovim:**

Open Neovim:

```bash
nvim
```
`lazy.nvim` (the plugin manager) will automatically install itself and all other plugins. This might take a moment on the first run. If you encounter any errors, please raise an Issue on the [repository](https://github.com/jesusplasencia/bolt.nvim). If no errors, press `q` or `:q` to quit the Lazy window.

## üöÄ Usage

`Bolt.nvim` is designed to be intuitive. Here are some basic commands and features to get you started:

*   **Which Key:** Show all the possible commands `<leader>`.
*   **Leader Key:** Your leader key is set to `<space>`.
*   **Toggle File Explorer:** `<leader>fe` 
*   **Find Files (Fuzzy):** `<leader>ff`
*   **Find Text in Files (Ripgrep):** `<leader>fg`
*   **Switch Buffers:** `<leader>b{buffer number}` e.g `<leader>b2`
*   **Save File:** `:w` (The classic way sport, there is no keymap for this)
*   **Quit Neovim:** `:q` (Come on! We are doing the classic way)
*   **Enter Insert Mode:** `i`
*   **Exit Insert Mode:** `<Esc>` or `<C-[>`

Explore the `lua/config/keymaps.lua` file for a complete list of keybindings.
