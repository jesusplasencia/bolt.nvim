## üõ†Ô∏è Requirements

Before installing `Bolt.nvim`, ensure you have the following installed:

*   **Neovim v0.10.0+ (or latest stable):** Required for optimal performance and Lua features.
    *   Check your Neovim version with `:version`
*   **`git`:** For cloning the repository and managing plugins.
*   **`python3` and `pip`:** Many Neovim plugins, including `ripxorip/bolt.nvim`, rely on Python3.
    *   Install `pynvim` for Neovim's Python3 interface:
        ```bash
        pip3 install pynvim
        ```
*   **`ripgrep`:** (Highly Recommended) Essential for fast and powerful search capabilities within Neovim and often used by fuzzy finders.
*   **A Nerd Font (Optional but Recommended):** For proper display of icons and glyphs (e.g., [Fira Code Nerd Font](https://www.nerdfonts.com/font-downloads)).

## üì¶ Installation

**1. Backup your existing Neovim configuration (optional but recommended):**

```bash
mv ~/.config/nvim ~/.config/nvim_old_backup
mv ~/.local/share/nvim ~/.local/share/nvim_old_backup
```

**2. Clone this repository:**

```bash
git clone https://github.com/jesusplasencia/bolt.nvim ~/.config/nvim
```

**3. Launch Neovim:**

Open Neovim:

```bash
nvim
```
`lazy.nvim` (the plugin manager) will automatically install itself and all other plugins. This might take a moment on the first run. If you encounter any errors, restart Neovim.

## üöÄ Usage

`Bolt.nvim` is designed to be intuitive. Here are some basic commands and features to get you started:

*   **Leader Key:** Your leader key is set to `<space>`.
*   **Toggle File Explorer:** `<leader>fe` (e.g., using `ripxorip/bolt.nvim` as the file manager)
*   **Find Files (Fuzzy):** `<leader>ff` (e.g., using Telescope.nvim)
*   **Find Text in Files (Ripgrep):** `<leader>fg` (e.g., using Telescope.nvim)
*   **Switch Buffers:** `<leader>fb`
*   **Save File:** `<leader>w`
*   **Quit Neovim:** `<leader>q`
*   **Enter Insert Mode:** `i`
*   **Exit Insert Mode:** `<Esc>`

Explore the `lua/config/keymaps.lua` file for a complete list of keybindings.
